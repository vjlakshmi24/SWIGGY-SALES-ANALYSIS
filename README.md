# SWIGGY-SALES-ANALYSIS
This Python-based data analysis project focuses on exploring and visualizing food delivery sales data to uncover business insights.

# 🍽️ Food Sales Data Analysis (Python Project)

## 📌 Problem Statement

The food delivery industry generates large volumes of transactional data including sales revenue, customer ratings, order frequency, and regional performance. However, raw data alone does not provide actionable insights for business growth.

The objective of this project is to analyze food order data using Python to identify sales trends, customer behavior patterns, and regional performance variations. By transforming raw data into meaningful insights and visualizations, this project supports data-driven decision-making.

---

## 📊 Project Overview

This project performs Exploratory Data Analysis (EDA) on food order data to uncover key business insights.

### 🔹 Overall Performance Metrics
- Total Sales (₹)
- Average Rating
- Average Order Value (₹)
- Ratings Count
- Total Orders

### 🔹 Time-Based Analysis
- Monthly Sales Trend
- Daily Sales Trend
- Weekly Trend Analysis
- Quarterly Performance Summary

### 🔹 Category & Regional Insights
- Total Sales by Food Type (Veg vs Non-Veg)
- Total Sales by State (Map Visualization)
- Top 5 Cities by Sales

---

## 🛠️ Tools & Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 📈 Key Insights

- Identified peak sales months and high-performing quarters.
- Analyzed weekday vs weekend sales patterns.
- Compared revenue contribution between Veg and Non-Veg categories.
- Determined top-performing cities and states.
- Evaluated customer satisfaction through rating analysis.

---

## 🚀 Conclusion

This project demonstrates how Python-based data analysis can transform raw food delivery data into meaningful business insights. The findings can help optimize marketing strategies, improve regional targeting, and enhance overall revenue performance.
